# Desktop Apps / Aplicaciones de Escritorio

## Español
Conjunto de aplicaciones de escritorio desarrolladas en C# como parte
de evaluaciones académicas. Demuestra el uso de interfaces gráficas
y lógica de programación.

### Tecnologías
- C#
- Windows Forms
- Visual Studio 2022

### Cómo ejecutar
1. Clona el repositorio
2. Abre el archivo `.sln` en Visual Studio
3. Presiona `F5` para ejecutar

---

## English
Set of desktop applications developed in C# as part of academic
evaluations. Demonstrates the use of graphical interfaces
and programming logic.

### Technologies
- C#
- Windows Forms
- Visual Studio 2022

### How to run
1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Press `F5` to run
